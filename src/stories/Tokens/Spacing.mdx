# Spacing Tokens

Spacing tokens provide consistent spacing values throughout the design system. All spacing tokens follow the `--gl-spacing-*` naming convention.

## Spacing Scale

| Token               | Value | Preview                                                                     |
| ------------------- | ----- | --------------------------------------------------------------------------- |
| `--gl-spacing-0000` | 0px   | <div style={{ width: '0px', height: '24px', backgroundColor: '#000' }} />   |
| `--gl-spacing-0025` | 2.5px | <div style={{ width: '2.5px', height: '24px', backgroundColor: '#000' }} /> |
| `--gl-spacing-0050` | 5px   | <div style={{ width: '5px', height: '24px', backgroundColor: '#000' }} />   |
| `--gl-spacing-0100` | 10px  | <div style={{ width: '10px', height: '24px', backgroundColor: '#000' }} />  |

| `--gl-spacing-0200` | 20px  | <div style={{ width: '20px', height: '24px', backgroundColor: '#000' }} />  |
| `--gl-spacing-0300` | 30px  | <div style={{ width: '30px', height: '24px', backgroundColor: '#000' }} />  |
| `--gl-spacing-0400` | 40px  | <div style={{ width: '40px', height: '24px', backgroundColor: '#000' }} />  |
