# Typography Tokens

Typography tokens define font families, sizes, weights, line heights, and letter spacing for consistent text styling throughout the design system.

## Font Families

- **AdihausDIN** - Primary functional font
- **AdineuePRO** - Standard/expressive font
- **AdihausDIN Cn** - Condensed expressive font
- **adineue PRO Cond** - Condensed standard font
